# Kalendarz Słowiański 2026-2040

Repozytorium zawiera plik ICS `kalendarz_slowianski_2026_2040.ics` z autentycznymi słowiańskimi świętami pogańskimi opartymi na astronomicznych wydarzeniach z lat 2026-2040.

## Główne cechy

- 8 głównych corocznych słowiańskich świąt związanych z przesileniami, równonocami i punktami środkowymi
- Precyzyjne czasy astronomiczne w UTC
- Kompatybilne ze wszystkimi standardowymi aplikacjami kalendarzowymi

## Użycie

1. Pobierz `kalendarz_slowianski_2026_2040.ics`
2. Zaimportuj do kalendarza:
   - **Google Calendar**: Ustawienia → Dodaj kalendarz → Z pliku/URL
   - **Outlook/Apple**: Plik → Importuj
